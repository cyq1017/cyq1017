### 3. 工作流 Skills（`.claude/commands/` 或 `.agents/skills/`）

用 Claude Code 的 slash commands 实现 8 个文科博士专属工作流：

| 工作流 | 命令 | 功能 |
|---|---|---|
| **晨间规划** | `/开始新的一天` | 回顾昨日进展，推荐今日重点（哪篇论文该推进？哪篇文献该读？） |
| **文献精读** | `/精读文献` | 引导式文献阅读：提取论点、方法、贡献、与自己研究的关系 |
| **概念溯源** | `/探索概念 <概念名>` | 深度梳理学术概念：定义、演变、代表人物、相关理论 |
| **论文推进** | `/推进论文` | 将想法转化为结构化论文项目（选题→大纲→章节） |
| **学术写作** | `/写作辅助` | 段落润色、论证逻辑检查、学术用语建议 |
| **读书笔记** | `/整理笔记` | 将非结构化阅读笔记整理入 vault |
| **归档整理** | `/归档` | 清理已完成项目 |
| **答疑解惑** | `/问一下 <问题>` | 快速学术问答 |

#### 每个 Skill 包含的文件：
```
.agents/skills/<skill-name>/
├── SKILL.md           # 详细工作流指令
└── examples/          # 示例（可选）
```
